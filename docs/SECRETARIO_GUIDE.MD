📘 Guía del Secretario de Cámara — cerebro_legislativo
Versión: 1.0
Última actualización: Mayo 2026
Para: Secretario(a) de Cámara Legislativa
Propósito: Guiarte paso a paso en el uso de la aplicación para gestionar sesiones, oficios, leyes y generar transparencia ciudadana.
🎯 ¿Qué es Cerebro Legislativo?
Es tu herramienta digital para:
✅ Gestionar sesiones, actas, oficios y leyes desde tu computadora
✅ Organizar la información legislativa sin depender de internet
✅ Sincronizar automáticamente un portal público para que los ciudadanos consulten leyes
✅ Generar códigos QR que permiten a cualquier persona descargar documentos oficiales
✅ Proteger tus datos con backup local y opcional en la nube
💡 No necesitas ser experto en tecnología. Si sabes usar Word y navegar por internet, puedes usar esta aplicación.
🚀 Inicio Rápido: Checklist de Primer Uso
□ 1. Descargar e instalar la aplicación (página 3)

□ 5. Sincronizar con el portal público (página 12)
□ 6. Generar tu primer QR institucional (página 14)
□ 7. Hacer tu primer backup (página 16)
📦 1. Instalación Paso a Paso
Para Windows (.exe)
1. Descarga el archivo: cerebro_legislativo-v1.0.0.exe
2. Haz doble clic para iniciar el instalador
3. Si Windows muestra "Windows protegió su PC":
   → Haz clic en "Más información"
   → Luego en "Ejecutar de todas formas"
4. Elige carpeta de instalación (recomendado: dejar la predeterminada)
5. Espera a que termine la instalación (~1 minuto)
6. Marca "Ejecutar Cerebro Legislativo" y haz clic en "Finalizar"
Para macOS (.dmg)
1. Descarga el archivo: cerebro_legislativo-v1.0.0.dmg
2. Haz doble clic para abrir el disco
3. Arrastra el ícono de la app a la carpeta "Aplicaciones"
4. Abre "Aplicaciones" y haz doble clic en "Cerebro Legislativo"
5. Si macOS muestra "No se puede verificar el desarrollador":
   → Ve a Preferencias del Sistema > Seguridad y Privacidad
   → Haz clic en "Abrir de todas formas" junto al mensaje de la app
6. La aplicación se abrirá normalmente
Para Linux (.AppImage)
1. Descarga el archivo: cerebro_legislativo-v1.0.0.AppImage
2. Abre la terminal en la carpeta de descarga
3. Ejecuta: chmod +x cerebro_legislativo-v1.0.0.AppImage
4. Ejecuta: ./cerebro_legislativo-v1.0.0.AppImage
5. (Opcional) Para integrar al menú: copia el archivo a ~/Aplicaciones/
⚠️ Nota importante: La aplicación funciona sin internet. Solo necesitas conexión para sincronizar con el portal público o hacer backup en la nube.

📅 4. Gestionar Sesiones Legislativas
Crear una Nueva Sesión
1. Haga clic en [📅 Sesiones] en el menú superior
2. Haga clic en [➕ Nueva Sesión]
3. Complete el formulario:

   ┌─────────────────────────────────┐
   │ 📅 Nueva Sesión                 │
   │                                 │
   │ Tipo: [Ordinaria ▼]             │
   │         • Ordinaria             │
   │         • Extraordinaria        │
   │         • Especial              │
   │                                 │
   │ Fecha y hora: [15/05/2026 09:00]│
   │                                 │
   │ Orden del día:                  │
   │ [___________________________]   │
   │ [___________________________]   │
   │ (Escriba cada punto en una línea)│
   │                                 │
   │ Comisión responsable (opcional):│
   │ [Salud ▼]                       │
   │                                 │
   │ [ Guardar Sesión ]  [ Cancelar ]│
   └─────────────────────────────────┘

4. La sesión aparecerá en la lista con estado "Pendiente"
Registrar Acta de Sesión
1. En la lista de sesiones, haga clic en la sesión deseada
2. Haga clic en [📝 Redactar Acta]
3. Use el editor para escribir el acta:
   • Formato básico: negrita, cursiva, listas
   • Puede pegar texto desde Word
4. Al finalizar, haga clic en [✅ Marcar como Firmada]
5. (Opcional) Adjunte PDF escaneado del acta física:
   • Haga clic en [📎 Adjuntar Documento]
   • Seleccione el archivo PDF
   • El sistema calculará automáticamente un hash de verificación

💾 9. Backup y Recuperación de Datos
Hacer Backup Local (Recomendado Semanalmente)
1. Haga clic en [⚙️ Ajustes] > [Backup y Restauración]
2. Haga clic en [💾 Exportar Backup Cifrado]
3. Ingrese su contraseña de Secretario (para cifrar)
4. Elija dónde guardar el archivo:
   • Carpeta local de su computadora
   • Memoria USB externa
   • Carpeta de red institucional
5. El archivo se nombrará:
   cerebro-backup-20260515-1430.enc
6. Guarde este archivo en un lugar seguro
🔐 Importante: El backup está cifrado. Sin su contraseña, nadie podrá restaurarlo. Guarde la contraseña por separado del archivo.
Restaurar desde Backup (Si cambia de computadora o hay fallo)
1. Instale Cerebro Legislativo en la nueva máquina
2. En la pantalla de inicio, haga clic en [🔄 Restaurar desde backup]
3. Seleccione el archivo .enc que guardó previamente
4. Ingrese la contraseña que usó al crear el backup
5. El sistema validará integridad y restaurará:
   • Todas sus sesiones, oficios y leyes
   • Configuración institucional
   • Preferencias de usuario
6. Reinicie la aplicación para completar
Backup en la Nube (Opcional - Google Drive)
1. En [⚙️ Ajustes] > [Backup y Restauración]
2. Active "Backup automático en Google Drive"
3. Autorice el acceso a su cuenta de Google (solo carpeta específica)
4. Configure frecuencia: [Diaria] [Semanal] [Al cerrar app]
5. El sistema subirá backups cifrados automáticamente
6. Para restaurar desde la nube:
   • Elija "Restaurar desde Google Drive"
   • Autorice acceso
   • Seleccione el backup deseado por fecha
⚠️ Precaución: El backup en la nube es opcional. Si prefiere mantener todo local, desactive esta opción. Sus datos nunca se comparten sin su consentimiento.
❓ 10. Preguntas Frecuentes y Solución de Problemas
🔐 "Olvidé mi contraseña de Secretario"
1. En la pantalla de login, haga clic en "¿Olvidó su contraseña?"
2. Ingrese su usuario
3. Responda la pregunta de seguridad que configuró
4. Si la respuesta es correcta, verá su código de recuperación (ej: A7K9-M2P4)
5. Ingrese el código para establecer una nueva contraseña
6. Si no recuerda la respuesta de seguridad:
   • Contacte al soporte técnico institucional
   • Proporcione su documento de identidad para verificación
🌐 "La sincronización con el portal no funciona"
Posibles causas y soluciones:

❌ Token de GitHub expirado o inválido
✅ Solución: 
   • Vaya a [⚙️ Ajustes] > [Sincronización]
   • Haga clic en "Renovar token"
   • Genere uno nuevo en GitHub y péguelo

❌ Sin conexión a internet
✅ Solución:
   • Verifique que su computadora tenga internet
   • La app reintentará automáticamente cuando se restablezca

❌ Repositorio de GitHub no encontrado
✅ Solución:
   • Verifique que el nombre del repositorio sea correcto
   • Asegúrese de que el token tenga permisos de escritura ("repo")

❌ Cuota de GitHub Actions excedida (raro)
✅ Solución:
   • Espere a que se renueve la cuota (cada hora)
   • O contacte a soporte para ajustar configuración
📱 "El QR no funciona en el celular de un ciudadano"
Posibles causas:

❌ El celular no tiene conexión a internet
✅ Solución:
   • El portal público requiere internet para cargar
   • Sugiera al ciudadano usar datos móviles o WiFi

❌ El navegador bloquea la descarga de Drive
✅ Solución:
   • El portal muestra un enlace alternativo visible
   • Indique al ciudadano: "Si no descarga, haga clic en 'Acceder directamente a Google Drive'"

❌ El código QR está dañado o muy pequeño
✅ Solución:
   • Imprima QR con tamaño mínimo de 2.5x2.5 cm
   • Use papel de buena calidad, evite dobleces o manchas
💾 "Quiero mover la app a otra computadora"
Siga estos pasos:

1. En la computadora actual:
   • Vaya a [⚙️ Ajustes] > [Backup y Restauración]
   • Haga clic en [💾 Exportar Backup Cifrado]
   • Guarde el archivo .enc en USB o nube

2. En la nueva computadora:
   • Instale Cerebro Legislativo (ver sección 1)
   • En la pantalla de inicio, elija [🔄 Restaurar desde backup]
   • Seleccione el archivo .enc e ingrese su contraseña

3. ¡Listo! Todos sus datos y configuración estarán disponibles.
🗑️ "¿Cómo elimino una ley u oficio por error?"
La aplicación no permite eliminación permanente inmediata (por seguridad).

Para "archivar" un elemento:
1. Abra el registro que desea ocultar
2. Cambie su estado a "Derogado" (leyes) o "Anulado" (oficios)
3. El elemento desaparecerá de las vistas principales
4. Para verlo nuevamente: use el filtro "Mostrar archivados" en la búsqueda

> 📌 Nota: Los elementos archivados se mantienen para trazabilidad legal, pero no aparecen en el portal público.

📚 12. Glosario de Términos
Término
Significado Simple
Backup
Copia de seguridad de sus datos para recuperarlos si algo falla
Cifrado
Proteger información con una contraseña para que solo usted pueda leerla
Hash
Código único que verifica que un archivo no ha sido alterado
Portal público
Página web donde los ciudadanos consultan leyes y legisladores
QR (Código de Respuesta Rápida)
Cuadrado con patrón que los celulares leen para abrir enlaces
Sincronización
Proceso de actualizar el portal público con los cambios que hace en la app
Token
Clave de acceso temporal para conectar la app con servicios externos (ej: GitHub)
Offline-first
La aplicación funciona completo sin internet; la conexión es opcional para funciones extra

📄 Anexo: Especificaciones Técnicas (Para Personal de TI)
• Aplicación: Electron 28 + React 18 + Vite 5
• Base de datos: SQLite con Drizzle ORM
• Almacenamiento local: %APPDATA%/cerebro_legislativo (Windows) o ~/.config/cerebro_legislativo (Linux/macOS)
• Backup cifrado: AES-256-GCM + PBKDF2 (100,000 iteraciones)
• Sincronización: GitHub REST API con cola offline y reintentos exponenciales
• Portal público: HTML/CSS/JS estático alojado en GitHub Pages
